# Лабораторная работа №24

## Вариант 24: Поиск пути в лабиринте (3)

Дан клеточный лабиринт на плоскости. Необходимо найти путь от заданной точки до выхода из лабиринта.

## Файлы
- main.h — заголовочный файл с определениями структур и прототипами функций.
- main.c — реализация операций.
- test.c — юнит-тесты.

- Символы:
  - `' '` — проходимая клетка
  - `'X'` — стена (непроходимо)
  - `'S'` — старт (может быть в любом месте)
  - `'E'` — выход (может быть несколько, выбирается ближайший)

Реализовано с использованием поиска в ширину (BFS).
